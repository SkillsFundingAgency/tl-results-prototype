Registration data format and rules

These are the technical requirements for Registration data. Please follow these instructions to enable you to upload a file to the Manage T Levels Results service.

In order to be accepted the file must:

  • be in CSV format
  • contain all mandatory fields
  • be a minimum of 1 or a maximum of 10,000 registrations

Add a new registration

Enter all mandatory data items for a ULN. If this ULN already exists we will overwrite the data we hold with this new data.

Amend a new registration

Enter all mandatory data items for an existing ULN. If we find this ULN we will overwrite the data that we hold. If this ULN does not exist we will add it as a new registration.

T Levels registration data: format and rules

Find out about the format and rules for registration data, how to add and amend registrations, and view an example CSV file.

Add a new registration

Enter the Unique Learner Number (ULN) and all mandatory data items. If this ULN already exists we will overwrite the data we hold with this new data.

Amend an existing registration

Enter all mandatory data items for an existing ULN. If this ULN already exists we will overwrite the data that we hold. If this ULN does not exist we will add it as a new registration.

Technical requirements

We can only accept files that:

  • are in CSV format
  • include all mandatory fields
  • contain a minimum of one registration
  • contain a maximum of 10,000 registrations

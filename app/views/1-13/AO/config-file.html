Configuration file format and rules

In order to be accepted the file must:

  • be in CSV format
  • contain all mandatory fields
  • be a maximum of 10,000 registrations

Add a new registration

Upload a Unique Learner Number (ULN) with the mandatory data items and a new record will be created.

Change an existing registration

Upload an existing ULN and the existing data we hold will be overwritten.

Delete an existing registration

Enter 'Yes' in the delete column and the ULN will be removed.

The config file must contain the following data items in this order:
Data item Format Example Mandatory Notes
ULN 10 digits 1002345567 Yes Unique Learner Number
UKPRN 8 digits 10010023 Yes UK Provider Reference Number
Core 2 digits 01 Yes Core codes
Specialism 1 3 digits 001 No Specialism codes
Must be submitted by 31/08/2020
Only one specialism is mandatory
Specialism 2 3 digits 001 No Specialism codes
Must be submitted by 31/08/2020
Only one specialism is mandatory
Delete Yes or blank Yes Use to delete an existing registration
Enter Yes to delete or leave blank

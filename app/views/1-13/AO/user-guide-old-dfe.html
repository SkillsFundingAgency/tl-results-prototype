The Manage T Level results service is run by the Education and Skills Funding Agency (ESFA), an executive agency of the Department for Education (DfE).

As an awarding organisation, through this service you can:

  • tell us if T Level details are correct
  • tell us which T Levels each provider is offering
  • upload, download and amend student registrations

This user guide tells you how to:

  • sign in and manage your account
  • manage your T Level details, providers and registrations

Signing in

In order to use Manage T Level results you need a DfE Sign-in account. You need to sign in with the work email address and password you set up for DfE Sign-in.

If you do not have a DfE Sign-in account, ask someone with an ‘approver’ role within your organisation to set up an account for you and give you access to Manage T Level results. DfE Sign-in will send you an email so you can verify your work email address.

Signing out

Select ‘sign out’ in the top right-hand corner of the screen to sign out and return to the introduction page. For security reasons you should always sign out when you have finished using the service.

Timeout

If you are signed in but are inactive for 20 minutes you will be signed out.

Changing your password

You can change your password at any time in your profile. If you forget your password you can reset it using the forgotten your password page.

Changing your email

You can change your email address at any time in your profile. Once you have changed your email address you need to activate it on a verification email. For security you also need to request the email address change on an email sent to your old account.

Finding the login page

You can visit the service at Manage T Level results.

Awarding organisation roles and permissions

The service roles you can be given are:

  • Site Administrator – has all roles: T Levels Reviewer, Providers Editor, Registrations Editor
  • T Levels Reviewer – reviews T Level Technical Qualification titles and confirms that details are correct or not
  • Providers Editor – can enter and amend the T Levels each provider is offering
  • Registrations Editor – can upload, amend and delete student registrations

A user can be given more than one role.

T Levels Reviewer tasks

From the Manage T Levels results home page if you select ‘T Levels’ the service will ask you to review the Technical Qualification details of any T Levels that have not yet been confirmed. This happens automatically for any T Levels awaiting confirmation.

In your review you should check that the Technical Qualification details are correct for each of your T Levels. This includes the T Level’s core component and specialisms. This is important because these details will appear on students’ certificates.

You do not need to review the route title. If all the details are correct you should select ‘yes’. You will then get a message to make it clear that the T Level’s Technical Qualification details have been confirmed.

When you have confirmed all your T Levels’ Technical Qualification details you can still view them to check everything remains correct.

If during your review you find any Technical Qualification details are wrong or missing, select ‘no’ to query it with us. We will then investigate your query and update you within 3 working days

You can still query a confirmed T Level’s Technical Qualification details at a later date, if you think the information is not accurate anymore

Providers Editor tasks

From the Manage T Level results home page if you select ‘providers’ you can tell us which T Levels a provider is offering. You find providers by searching using their name. You can then select which T Levels to add or remove. We recommend that you only add a T Level to a provider once the T Level’s details have been confirmed as correct in the ‘T Levels’ section.

Registrations Editor tasks

From the Manage T Level results home page if you select ‘registrations’ you can upload and amend student registrations in bulk through a data file. You can also add and cancel single registrations manually.

Find out more about the technical requirements for the file in our registration data and format rules guide.

Get help with Manage T Level results

If you have any general questions about using our service you can contact us.

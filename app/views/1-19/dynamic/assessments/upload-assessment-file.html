Upload multiple assessments

You can add or change assessment entries for more than one learner by uploading CSV files.

Submit

Before you start

If you export data from another system, you must check the data is in the right format. You can download a format and rules guide to help you.

Download T Levels assessment entry data format and rules guide (XLSX, 158KB)

The ULN must have an active registration status. You cannot upload an assessment entry for a withdrawn learner.

If a core or occupational specialism assessment entry is already held and a blank one is submitted in the file, the existing entry will be deleted.

If the data is not correctly formatted, nothing will be uploaded. You will be able to download an error report which tells you what's wrong.

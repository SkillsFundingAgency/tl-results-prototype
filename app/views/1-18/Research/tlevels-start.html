Manage T Level results

This service is for awarding organisations and providers who deliver T Level qualifications.

Awarding organisations (also known as exam boards) can use this service to:

  • tell us if T Level details are correct
  • tell us which T Levels each provider is offering
  • add and amend learner registrations
  • add and amend assessment entries
  • add and amend core component results
  • access the Manage T Level results user guide

Providers (for example, schools and colleges) can use this service to:

  • tell us if a learner has achieved the minimum standard for English and maths
  • tell us if a learner has completed their industry placement
  • request statements of achievement

You will be signed out if you are inactive for 30 minutes.

Before you start

You will need a DfE Sign-in account and an assigned service role.

Awarding organisation service roles

T Levels Reviewer – reviews T Level Technical Qualification titles and confirms that details are correct or not

Providers Editor – can enter and amend the T Levels each provider is offering

Registrations Editor – can upload, amend, view, remove and add learner registrations and assessment entries

Results Editor – can upload, amend, view, remove and add learners’ core component results

Site Administrator – has all roles: T Levels Reviewer, Providers Editor, Registrations Editor, Results Editor

Provider service roles

Learner Records Editor – can upload, amend, view, delete and add English and maths achievements and update industry placement completion status

Provider Administrator – has all roles: Learner Records Editor
